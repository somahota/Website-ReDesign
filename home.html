Responsive Re-Design

Design: GatesNFences.com

A "responsive redesign" focussed on cleaning and revamping the gatesnfences website to improve user usability.

Introduction

I have chosen the gatesnfences.com website. This is a site for users to buy different types of gates and/or fences for their needs. This site has a large range of items available, however, there are some critical issues with the website design and my aim is to redesign the website fixing those issues.

Part 1: Identifying Usability Problems

I have conducted accessibility-checking using the WAVE service tool

As we can see from the summary image, there are 2 main problems associated with this site.

  • Missing alternative text errors
  • Contrast errors

Additionally, there are a few other problems that I can see with the design layout of the website and the website content.

When I evaluated these errors with respect to user readability,memorability, and usability, the following table discusses how these errors contribute to the mentioned user problems.

  • Contrast

    • The bright red colored title in the green background is difficult to read.
    The gray colored background with the blue and red text is difficult to read.

    The bright pink marble colored background with red colored text is very difficult to read.
    The bright pink marble colored background with red colored text and red colored buttons is very difficult to understand .

  • Website Layout

    • The address part should not be located in the header. Also, adding a text on the header urging users to scroll down the page to find their work on Disney World looks inappropriately placed.
    The layout of the buttons are vertical and the other tabs are horizontal. So there is an inconsistency with the way they are laid out.
    The categories of the Gates, Fences, and the Operators are not classified together in an orderly manner. They are all displayed vertically one below the other. As a user, it is very difficult for me to quickly find what I want and navigate around.
  • Website Content


    • This is a very big block of text. As a user, I might not only find it overwhelming to read such a huge block of text but also feel confused as to where to begin from.

    Overall, this website does not seem user friendly in terms of accessibility and usability. The user will feel overwhelmed and confused with the confusing layout of tabs and buttons thus causing problems with learnability. Additionally, the color choices add on to the confusion by decreasing readability.

Part 2: Visual ReDesign

My goal is to simplify this complex looking website so that user's find it easy to understand,use and read.
